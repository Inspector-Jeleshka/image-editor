Описание программы
Программа позволяет редактировать изображения формата BMP с помощью консольного интерфейса. Вы можете выполнять следующие действия: поворот изображения, обрезка, изменение отдельных пикселей, создание коллажа, добавление текста, изменение яркости и контраста.

Доступные команды:
1. Rotate image:
Описание: Поворачивает изображение против часовой стрелки на заданный угол.
Требуемые данные:
Угол поворота (в градусах).
Координаты центра поворота (x, y).
2. Crop image:
Описание: Обрезает изображение по двум заданным угловым точкам.
Требуемые данные:
Координаты первой точки (x1, y1).
Координаты второй точки (x2, y2).
3. Change pixel:
Описание: Изменяет цвет пикселя в заданной точке.
Требуемые данные:
Координаты пикселя (x, y).
Новые значения цвета (R, G, B).
4. Make collage:
Описание: Создаёт коллаж, накладывая второе изображение поверх текущего.
Требуемые данные:
Путь до второго изображения (абсолютный или относительный).
Координаты размещения (x, y).
5. Add title on top of image:
Описание: Добавляет текст на изображение.
Требуемые данные:
Текст для добавления.
Координаты начальной точки текста (x, y).
6. Edit image brightness:
Описание: Изменяет яркость изображения.
Требуемые данные:
Значение изменения яркости (целое число; положительное — увеличить, отрицательное — уменьшить).
7. Edit image contrast:
Описание: Изменяет контраст изображения.
Требуемые данные:
Значение изменения контраста (число с плавающей точкой; допустимый диапазон: от -254 до 258).
8. Save:
Описание: Сохраняет текущее изображение в файл.
9. Exit:
Описание: Выход из программы без сохранения.

Примечания
Программа поддерживает только изображения формата BMP с глубиной цвета 24 бита.
Убедитесь, что вводимые данные корректны (например, координаты находятся в пределах изображения).
Если файл с изображением повреждён или неподдерживаемого формата, программа уведомит вас об ошибке.
